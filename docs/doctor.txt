Tool and kernel verification

Checks if all required external tools are installed and verifies kernel
support for optional features like the eBPF sandbox.

Usage:
	desk doctor

Required tools:
	docker		Container runtime
	k3d		k3s cluster management
	kubectl		Kubernetes CLI
	helm		Kubernetes package manager

Kernel checks:
	BPF LSM		Required for the eBPF sandbox DaemonSet

The command will display the status and version of each tool, along with
installation instructions for any missing tools. Kernel checks are shown
as warnings and do not cause the command to fail, since the eBPF sandbox
is optional.

If BPF LSM is not available, see docs/macos-ebpf-lima.md for instructions
on running the eBPF sandbox on macOS using Lima.
